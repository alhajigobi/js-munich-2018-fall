Remember query selectors?

  • Where did we use query selectors?

    In CSS to style specific elements

  • What does this select? 
    
                            .main p

    All paragraphs inside elements of class main

  • What does this select? 
    
                            ul.infotable > li:last-child

    The last direct list item of every UL-element with class "infotable".

Query selectors:

  • are very powerful.
  • are great for finding specific elements in a HTML page.
  • Why not use them in JS?

document.querySelector('#footer a')

Returns the first link in the element with id "footer".


document.querySelectorAll('#footer a')

Returns all links in the element with id "footer".
